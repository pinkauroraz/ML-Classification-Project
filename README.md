# Machine Learning Classification Project

This project was completed as part of **TUMO Labs – Machine Learning Learning Station 2**.

## About

In this project, I explored several machine learning classification algorithms and compared how they perform on the same dataset. The goal was to better understand the classification workflow, from preparing the data to evaluating different models.

## Dataset

The project uses the **Breast Cancer Wisconsin (Diagnostic) Dataset**, available through Scikit-learn.

- 569 samples
- 30 numerical features
- Binary classification:
  - 0 – Malignant
  - 1 – Benign

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Models

The following models were trained and evaluated:

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Workflow

- Load the dataset
- Explore the data
- Split the data into training and testing sets
- Scale the features
- Train the models
- Evaluate their performance
- Compare the results

## Evaluation

The models were compared using:

- Accuracy
- Confusion Matrix
- Classification Report
